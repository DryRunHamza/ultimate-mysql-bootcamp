## How to Select Database
 
 ```sql
 USE <Dbname> || use <Dbname> 
 USE mydb || use mydb
 ```

## How to Select all Table fields in Database using "SELECT *"

 - First Select Database.
 - Select table
 - Also use semicolon ; at the end of each statement.

 ```sql
 USE mydb; -- write your own database
 SELECT * 
 FROM customers;
 ```
